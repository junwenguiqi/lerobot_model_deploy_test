# 输入数据说明

ACT、LeRobot Diffusion Policy 和 LeRobot PI0/PI0.5 共用以下 canonical observation：

```python
{
    "observation.images.front": front_rgb_uint8,
    "observation.images.wrist": wrist_rgb_uint8,
    "observation.state": state_float32_7,
    "observation.tactile_left": tactile_left_1d,
    "observation.tactile_right": tactile_right_1d,
}
```

要求：

```text
front / wrist: HWC RGB uint8
state: float32[7]
state = [x, y, z, rotvec_x, rotvec_y, rotvec_z, gripper_width]
```

默认动作：

```text
action: float32[7]
[dx, dy, dz, d_rotvec_x, d_rotvec_y, d_rotvec_z, target_gripper_width]
```

动作是逐步相对末端增量：

```text
action_t = target_(t+1) relative to state_t
```

## LeRobot v3 数据集

至少需要：

```text
observation.images.front
observation.images.wrist
observation.state
action
timestamp
frame_index
episode_index
task_index
index
meta/info.json
meta/stats.json
```

触觉字段可以保留，但当前 vanilla ACT、DP 和 PI adapter 默认不消费触觉。

模型差异：

| 模型 | 图像大小 | 时间结构 | 备注 |
| --- | --- | --- | --- |
| ACT | 256×256 | 当前观测 + action chunk | mean/std normalization |
| LeRobot DP | 256×256 | observation history + horizon | min/max normalization |
| LeRobot PI0/PI0.5 | 224×224 | 当前观测 + task + action chunk | 需要语言 task；可能下载模型文件 |

训练和在线部署必须保持相同的图像颜色、state 顺序、action 语义和归一化统计。
